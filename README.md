# React-Fundamentals

## About

React 기본 작업 환경
- babel 을 통한 ES6 변환
- webpack-dev-server 사용
- react-hot-loader 를 통한 Hot Module Reload 사용

## 설치하기

```sh
git clone https://github.com/dev-chulbuji/react-fundamentals.git
npm install -g webpack webpack-dev-server
npm install
```

## 실행하기

```
npm run dev-server
```
